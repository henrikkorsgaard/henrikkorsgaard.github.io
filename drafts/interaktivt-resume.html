Deployment and organisational integration Rolling out a new system is far from trivial. I have experience with deploying applications to app stores and using common technologies like Docker to deploy applications on cloud infrastrucute. That is the easy part. The hard part is to strategically plan the the integration of the new system into the organisation. Do you pilot a subset of the system across the organisation, pilot the full system within a single unit, or do you go for the risky full tranition all at once?
