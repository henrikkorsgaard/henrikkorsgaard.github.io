Invoke the planner agent to help organize and plan work.

1. Ask what needs to be planned (if not specified)
2. Create or update a plan file in `/planning`
3. Break down into actionable steps
4. Identify any decisions needed from Henrik
5. WAIT for confirmation before any implementation

Plans should be saved as: `/planning/<topic>-plan.md`
